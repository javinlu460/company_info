# 企业官网系统开发计划

## 项目概述

参考 www.shandongzhongbang.cn 开发一套企业官网系统，采用前后端分离架构。

- **后端**：SpringBoot + MySQL + MyBatis-Plus + Spring Security + JWT
- **前台**：Vue 3 + Vite + Element Plus（或自定义样式）
- **后台管理**：Vue 3 + Vite + Element Plus
- **项目路径**：`d:\IdeaProjects\company_info`

---

## Task 1：后端项目初始化与基础架构

**目标**：搭建 SpringBoot 后端项目骨架

**内容**：
- Maven 多模块项目结构（common、system、admin、web）
- 数据库连接配置（MySQL）
- MyBatis-Plus 配置（分页插件、自动填充）
- Spring Security + JWT 认证鉴权
- RBAC 权限模型（用户、角色、菜单/权限表）
- 全局异常处理、统一响应封装
- 文件上传（本地存储）配置
- Swagger/Knife4j API 文档

**核心数据库表设计**：
```
sys_user        -- 用户表
sys_role        -- 角色表
sys_menu        -- 菜单/权限表
sys_user_role   -- 用户角色关联
sys_role_menu   -- 角色菜单关联
sys_config      -- 系统配置（公司名称、logo、联系方式等）
cms_product_category -- 产品分类
cms_product     -- 产品
cms_news_category    -- 新闻分类
cms_news        -- 新闻/文章
cms_banner      -- 轮播图
cms_message     -- 留言
cms_friendly_link    -- 友情链接
```

## Task 2：后端业务模块开发

**目标**：实现所有业务 API 接口

### 2.1 系统管理模块
- 用户管理 CRUD + 分配角色
- 角色管理 CRUD + 分配权限
- 菜单管理（树形结构）
- 系统配置管理（公司名称、logo、联系方式、地址、备案号等）

### 2.2 内容管理模块
- **产品中心**：产品分类 CRUD、产品 CRUD（支持富文本详情、多图上传）
- **新闻中心**：新闻分类 CRUD、新闻 CRUD（支持富文本编辑）
- **轮播图管理**：Banner CRUD（图片、链接、排序）
- **留言管理**：查看留言列表、标记已读、删除

### 2.3 前台展示接口
- 首页数据聚合接口（Banner、推荐产品、最新新闻、公司简介）
- 产品列表/详情接口（分页、按分类筛选）
- 新闻列表/详情接口（分页、按分类筛选）
- 联系信息接口
- 在线留言提交接口（含验证码）

---

## Task 3：前台 Vue 项目开发

**目标**：开发企业官网前台展示站

**页面结构**（参考原网站）：
1. **首页** - Banner轮播 + 产品展示 + 关于我们 + 新闻动态 + 在线咨询
2. **关于我们** - 公司简介、企业文化、资质荣誉、公司愿景（Tab切换）
3. **产品中心** - 分类筛选 + 产品卡片网格展示 + 产品详情页
4. **新闻中心** - 新闻列表（带日期、摘要）+ 新闻详情页
5. **业务领域** - 应用场景展示
6. **联系我们** - 联系信息展示 + 在线留言表单

**设计风格**：
- 主色调：深蓝色（#003366），辅助色：天蓝色
- 工业企业风格，响应式布局
- 导航栏固定顶部，移动端可折叠

---

## Task 4：后台管理系统 Vue 项目开发

**目标**：开发后台 CMS 管理系统

**功能模块**：
1. **登录页** - JWT Token 认证
2. **仪表盘** - 数据统计概览（产品数、新闻数、未读留言数）
3. **系统管理** - 用户管理、角色管理、菜单管理
4. **网站配置** - 公司名称、logo、联系方式、关于我们内容、底部信息
5. **内容管理**：
   - 轮播图管理（拖拽排序、上传图片）
   - 产品分类管理（树形结构）
   - 产品管理（列表、新增/编辑含富文本编辑器）
   - 新闻分类管理
   - 新闻管理（列表、新增/编辑含富文本编辑器）
6. **留言管理** - 留言列表、查看详情、标记状态、删除
